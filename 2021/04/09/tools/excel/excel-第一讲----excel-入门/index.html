excel 入门

excel详解

基本操作

基本概念

excel 中每一个格子叫 单元格

下一个单元格，enter

右一个，tab

左一个，shift+tab

键盘的上下左右键 可以更简单控制

单元格宽度自适应

要想让单元格宽度和文字长度一致，可以双击 F 和 G 之间的间隔线。

合并后居中

将整一行的单元格合并以后，将文字放在中间

直接点击 菜单栏 –》 常用 –》 合并后居中 即可。

其他各种颜色，字体等等，都可以在这里找到。

将所有栏位宽度（高度）调整为一致

框选所有想要相同宽度的单元格，之后选中其中任何一个间隔线 拖动到想要宽度，之后所有单元格宽度都会一致。

列高 的操作也同样适用

产生等差（等比）数列

将鼠标移动到 T002 右下角，变成 + 后，往下拖动。可以自动生成数列。

等差，等比数列 也可以。

日期设置

excel 默认的日期格式以 / 作为分隔，方便进行后期的各种日期格式转换。例如 2021/04/09 。

输入完成以后，右键日期，可以进行各种格式的调整。

如果出现全部都是 ######## 符号。 原因是长度超过了单元格的长度。

可以双击右侧的分隔线，自适应内容长度，即可解决问题。

快捷键：

即可自动产生默认日期。 2021/04/09

当然也可以 拉住右下角往下拖, 来实现自动补充。并且可以进行如下选择。

计算

如上图所示，想要进行乘法计算，只需要在 总金额处：

  1. 写一个 = 符号
  2. 鼠标左键单击 45 所在的单元格
  3. 鼠标左键单击 30 所在的单元格
  4. 回车

如果要进行公式的修改，双击单元格，或者 到上方的编辑页直接编辑。

excel 进行的复制，是计算公式的复制，不只是值的复制

当然，也可以通过拖拉右下角来实现整列的计算。

价格的设置

选择整一列以后，在 菜单 –》 常规 中选择 货币，就可以将数字转变成货币形式。

同时可以根据自己需求，进行小数位数减少等的操作。

左上角有个小刷子（我的截图最左上角从上往下第三个），可以复制格式。选择了以后，选中其他的金额列就可以复制格式了

表格设计

表格边框

如果在打印中，没有表格线，框。可以有两个方法

默认表格形式

页面布局 –》 网格线 –》 打印 中的打勾，即可打印出表格边框。

自定义表格形式

  1. 关闭上面的 打印中的打勾。
  2. 开始（常用） –》边框选择 –》 线条颜色 ，线型
  3. 这时候，鼠标会变成画笔，可以 画横线，或者竖线，或者画框（只有外框）。
  4. 开始 –》 边框选择 –》 绘制边框网络 ，选择后，画笔会画出框以及内部所有网格。
  5. 按住 shift不放，会变成橡皮擦，可以擦除格式。
  6. esc 退出画笔功能。
  7. 还有另一种方式，是先选择范围，再进行边框的选择，和 开始 –》 边框选择 –》 上下框线 基本类似

表格背景图片

背景图片无法打印出来，但是能提高表格的美观程度

  1. 页面布局 –》 背景 中可以选择。

  2. 图片中存在的网格线，可以通过如下去除：页面布局 –》 网格线 –》 查看 ，把 √ 去掉。

  3. 如果有字部分因为图片原因难以辨认，可以先框选表格部分， 开始 –》 油漆桶 –》 选个白色即可。

  4. 选择 A 列，右键，选择插入，可以在右侧插入一列，更加美观。

  5. 框选一部分以后， 开始 –》 油漆桶 –》 无填充 ，又可以显示背景图片。选择颜色，就又可以覆盖。

冻结窗格

  1. 有时需要将表头冻结，使其在滚轮下滑时，固定在第一行

  2. 首先选中需要冻结的行的下一行

  3. 视图 –》 冻结窗格 ，即可冻结自己所想的那一行。

  4. 同时也能冻结列，同理，选择想要冻结的右一行，选择 冻结窗格即可。

分割视窗

  1. 分割视窗，常用来进行 **上下跨度比较大 ** 的两列之间的比较。比如 1月和 12月 的数据距离比较远，想资料比对，可以通过这个实现。
  2. 选中想要分割的 下一行，视图 –》 拆分
  3. 之后，上下两部分可以独立滚动。
  4. 如果将 分割线拖到表格最上方，就可以关闭 拆分功能。

套用表格格式

类似于word， excel有更强大的转化为表格的方法。

第一种，直接格式化

  1. 开始 –》 套用表格格式。
  2. 直接对表格样式进行更改。

第二种，用表格化

  1. 框选想要 修改表格的区域。
  2. 插入 –》 表格
  3. 会跳到 表格栏，便可以选择自己喜欢的样式。
  4. 镶边行，镶边列 等都可以设置。
  5. 如果觉得颜色不够多，可以到 页面布局 –》 颜色 。 整个表格都会变化颜色。
  6. 表格中的 汇总行 ，会将最后一列值相加。 可以拖动右下角，往左边拖，使每一列都计算出汇总数据。
  7. 同时对每一列的计算，都可以选择不同的计算公式。见下图

  1. 表格新增一行 ：在上一行选中一个单元格，按 tab 键可以新增一空白行

  2. 表格新增一列：在标题同一行新增一个单元格，回车即可。

  3. 清除所有样式：只要 表格 –》 选择样式处 –》 第一个无样式。

  4. 退出表格模式：表格 –》 转换为区域 ， 即可变成原始状态。

数据分析

排序

默认排序

  1. 选中想要排序的列 中的某一个单元格
  2. 开始 –》 排序和筛选 –》 升序 ， 即可升序排序
  3. 排序存在的问题：不能简单框选，然后进行排序。 可能会只排序那一列，其他几列没有排序，导致 各个月份的数据出现错乱。
  4. 所以不要框选整栏的资料
  5. 如果要进行多列的排序，则 先选择一个单元格
  6. 开始 –》 排序和筛选 –》 自定义排序 ，可以增加多列来进行排序。

自定义排序

  1. 根据自己定义的顺序来进行排序 ，常 用于中文等的排序
  2. 开始 –》 排序和筛选 –》 自定义排序 –》 顺序 –》 自定义列表
  3. 按需加入，就可以进行排序。
  4. 在第二步中的自定列表中，可以看到已经有部分值。
  5. 用处是， 在单元格中输入 一月 以后，按住右下角往下拉，可以出来已经配置的值。
  6. !!所以可以自定义所需的值，到时候往下拉，就可以出来该顺序的值。

筛选

普通筛选

可以隐藏不满足条件的数据。只显示满足条件的数据。

  1. 开始 –》 排序和筛选 –》 筛选 。
  2. 在每一列的表头就出现 向下的箭头，可以选择想要筛选的数据。
  3. 模糊筛选： 每一列表头的筛选中 –》 开头是 ， 就可以进行文字的筛选。
  4. 对数字 进行 高于，小于，介于 等的筛选也是可以。
  5. 取 前几项， 个数，百分比 也可以。
  6. 日期筛选 也有特定的筛选器，可以选择 某个日期之后，上个月，这一季 都可以。
  7. 色彩排序： 按住 ctrl（command），选择几个单元格 –》 油漆桶，选择喜欢的颜色。 –》 排序下箭头 –》 按颜色排序 。
  8. 色彩筛选当然，也可以对上一步中的 色彩进行筛选。

切片器

切片器，就是强化版的筛选。

  1. 在表格 –》 插入切片器 。选择想要生成的列的切片器。
  2. 多选几个后，会生成多个框。
  3. 点击其中的 栏目，可以快速进行筛选。
  4. 多选条件也是可以，按住 ctrl键不放，可以选择多个条件。

  1. 去除切片器：删除即可。

条件格式

筛选会将不符合条件的隐藏。

条件格式，会用特殊颜色来显示筛选结果。

  1. 选定一列
  2. 开始 –》 条件格式 –》 突出显示单元格规则 –》 等于 –》 输入一个值（或者直接用鼠标点击一个单元格作为条件）。
  3. 符合条件的值便会高亮。
  4. 日期，介于 等等条件，也是相同方法。
  5. 开始 –》 条件格式 –》 数据条 色阶 图标集 也可以更明显的显示数据的大小，各个规则也可以自定义。如下是数据条。

  1. 清除规则： 开始 –》 条件格式 –》 清除规则 –》 清除整张表的规则 。可以还原所有信息。

公式设定

工作表

excel 最大的特点，就是一个excel文件内可以有多个工作表

基本操作

  1. 复制工作表： 按住ctrl ，鼠标左键拖动便可以复制。
  2. 上述复制工作表也可以在不同的excel 之间复制。 工作表上–》右键 –》 移动或复制 –》 选择目标excel –》 建立副本。

工作表群组

同时操作多张工作表的内容和格式。

群组基本操作

  1. 按住ctrl，选择多张工作表
  2. 修改样式，会同步修改所有群组中的工作表
  3. 退出群组： 右键某一张工作表 –》 取消工作表群组。
  4. 隐藏：上述方法，右键之后，也可以隐藏，取消隐藏。

合并计算

  1. 可以将多张工作表内容合并，计算到同一张工作表。
  2. 数据 –》 合并计算 –》 函数（选择求和） –》 引用（点击具体的表，框选内容） –》 点击 + 号 新增 –》 如果有标题栏 等，可以选择最下面的 首行,最左列 –》 确定，即可将数据汇总。
  3. 上述步骤中，选择 创建指向源数据的链接 ，会在表格最左侧显示 + 号，可以查看是从哪张表格来源。
  4. 常常用于不同月份，不同年份的合计。

图表

表格转成图表

基本方法

  1. 选中表格，包含 行头,列头 , 因为他们可以作为坐标轴文字。

  2. 创建图表：插入 –》 建议的图表。

  3. 图表移动到新的工作表： 图表设计 –》 移动图表

  4. 图表内新增文字等项目： 图表设计 –》 添加图表元素 。 可以加入 坐标轴标题 等等。

  5. 纵坐标轴标题竖排：右键标题 –》 设置坐标轴标题格式 –》 文本选项–》 文字效果 –》 文字方向 –》 竖排

  6. 表格与图表同步：新增数据 –》 拖动单元格右下角 –》 使图标内容包含整个表格 。 当然也可以进行筛选。

  7. 更改图标样式：图标设计 –》 更改图标类型 。

  8. 切换行列：图标设计 –》 切换行/列。

中级表

  1. 横（纵）坐标的范围更改：坐标轴上右键 –》 设置纵坐标格式。 可以设置最大，最小值，主刻度，次要刻度等
  2. 用图片替代直方图：直方图上右键 –》 设置数据列格式 –》 油漆桶 –》 图片或纹理填充（可以用自己的图片）。
  3. 饼图：ctrl选中多块区域 –》 插入 –》 饼图
  4. 饼图中添加说明： 图表设计 –》 添加图表元素 –》 数据标签 –》 数据标签外 。 或者直接右边的 + 号。
  5. 饼图区块移出：双击选中某一块饼 –》 移除–》 设置阴影等。
  6. 折线图：插入 –》 折线图。
  7. 折线走势图（迷你图）：选中表格最右边空白栏位 –》 插入 –》 迷你图 –》 框选表格中数据 –》 确定。可以显示迷你折线图。

  1. 迷你图样式修改：迷你图 –》 设置柱形等，颜色等。
  2. 删除迷你图：迷你图 –》 清除
  3. 散点图：选中数据 –》 插入 –》 散点图 –》 右上角+号 –》 趋势图
  4. 雷达图，组合图 等，用法都类似。

数据透视表

  1. 选中一个单元格 –》 插入 –》 数据透视表

  2. 可以排序，筛选

  3. 可以将栏目直接拖拉到 筛选栏等里面。

  4. 数据透视表 == 积木玩具，可以根据自己想要的来操作。

  5. 单元格可以组成群组

  6. 各个栏目都可以重命名

  7. 数据透视图 ：数据透视表分析 –》 数据透视图

  8. 数据同步：原始表格更新了数据，需要同步数据： 数据透视表分析 –》 刷新

  9. 自动同步：表格创建时候，即为动态表格： 框选所有数据 –》 开始 –》 套用表格格式 –》 选择其中一个即可。

打印设定

分页设定

  1. 查看打印的分页：视图 –》 分页预览 。 虚线是分页线
  2. 调整分页：选中分页线 –》 拖拉即可。 可以水平拉（将最后一列总计放到正常页面中），可以竖直拉（按照月份来分页打印）。
  3. 手动插入（移除）分页：页面布局 –》 分隔符 –》 插入（移除）分页。
  4. 标题列在每一页中都显示：页面布局–》 打印标题 –》 顶端标题行 –》 选择标题行。
  5. 上述是第一种方式，下面是第二种方式
  6. 调整页面大小：页面布局 –》 宽度（高度，缩放比？）
  7. 打印特定行：页面布局 –》 打印区域 –》 设置打印区域（清除）

页眉页脚，水印

  1. 查看 页眉，页脚 ： 视图–》页面布局 。 每一个都分为 左中右 三个区块。
  2. 添加页码：选中页脚的右方格 –》 页眉和页脚–》 页码 –》 在旁边可以输入文字，比如 第 页 –》 就会显示 第一页
  3. 添加总页数：选中页脚有方格 –》 页眉和页脚 –》 页数 –》 添加文字 –》 第1页/共4页
  4. 页尾模板：页眉和页脚 –》 页尾 –》 选择模板
  5. 可以插入图片：页眉和页脚 –》 图片 –》 调整大小 –》 设定百分比
  6. 水印：空白工作表 –》 插入 –》 A（文字） –》 输入想要的文字（比如：机密） –》 调整文字大小，倾斜 –》 保存到 ppt 或者 word –》 保存成图片 –》 选中页眉中间框 –》 插入图片 –》 调整图片大小 和 刷淡 –》 最后成功。

函数

最基本的整列计算相加

  1. 开始 –》 自动求和 按钮，会自动猜测要相加的范围，生成表达式
  2. 也可以用鼠标自己来选择想要框选的范围。
  3. 也可以按住 ctrl 不放来选择多个范围。
  4. enter 键即可进行计算
  5. 其余的 平均值，最小值，最大值 等，与其基本类似
